Tic Tac Toe Game (Python)

This is a simple console-based Tic Tac Toe game written in Python.
The game is played between two players: Player X and Player O.

## Project Description

The game displays a 3×3 Tic Tac Toe board where players take turns placing their symbols.
After every move, the program checks whether a player has won or if the match has ended in a draw.
The game stops automatically once the result is decided.

## Features

* Two-player game (X vs O)
* Console-based board display
* Simple position-based input
* Win detection (rows, columns, diagonals)
* Draw detection
* Prevents invalid or repeated moves

## How to Run the Program

1. Make sure Python is installed on your system.
2. Open the project folder.
3. Run the file using:
   
   python main.py

## How to Play

* The board positions are numbered from 0 to 8
* Player X starts the game
* Players take turns entering a position number
* The game ends when a player wins or when it results in a draw


## Board Position Reference

| 0  | 1   | 2  |
| -- | --- | -- |
| 3  | 4   | 5  |
| 6  | 7   | 8  |

## Game Rules

* A player wins by placing three symbols in a row, column, or diagonal
* If all positions are filled without a winner, the match is a draw
* A position once filled cannot be used again
